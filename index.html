Introducing MathDict - A Mathematical Dictionary

During my first year, I faced challenges in finding the correct Sinhala and Tamil translations for mathematical terms. This inspired me to develop MathDict, a web application that provides Sinhala and Tamil meanings of mathematical terms, along with brief explanations and examples. MathDict was created as a solution to help students better understand mathematical terminology, with guidance and support from senior peers.
I'm excited to share this project and hope it benefits many students facing similar difficulties!


Introducing Our Smart Locker System!

We've developed a smart locker system web application that ensures secure and seamless access using auto-renewing QR codes (refreshing every 2 minutes!) and downloadable access options. Our system allows users to store, retrieve, and manage locker access through a digital interface. Key features include user authentication, real-time locker availability tracking, and secure QR code integration.
